# v2ray-route-config-generator
一个基于纯HTML的 V2rayN/路由规则配置生成器，支持自定义域名分流和一键兜底策略
# V2rayN/Xray 路由配置工具 V2.0

## 🎯 项目简介 (Project Overview)

本项目是一个轻量级、**纯前端**（HTML + JavaScript）的路由规则生成工具，旨在帮助 V2rayN 和 Xray 用户快速、直观地定制和生成复杂的路由分流配置 JSON 文件。

用户可以在网页上通过可视化界面添加高优先级的分流规则，并选择智能的**兜底策略**（如“大陆直连，国外代理”），最终一键复制 JSON 代码导入客户端。

## ✨ 核心功能

* **可视化配置：** 通过表单快速添加自定义域名和对应的出站策略（代理/直连）。
* **专业术语：** 使用专业的“代理 (Proxy)”和“直连 (Direct)”标签。
* **兜底策略：** 提供三种常用的默认分流策略，包括基于 `geosite:cn` 和 `geoip:cn` 的智能分流。
* **一键复制：** 自动将生成的格式化 JSON 代码复制到剪贴板。

## 🖥️ 如何使用 (How to Use)

### 方式一：在线预览 (推荐)

由于本项目是纯前端 $HTML$，您可以直接在任何支持 GitHub Pages 的服务上部署或在本地浏览器中运行 `index.html` 文件。

### 方式二：本地运行

1.  将 `index.html` 文件下载到您的本地电脑。
2.  使用任意现代浏览器（如 Chrome, Firefox, Edge）直接打开 `index.html` 文件即可开始使用。

### 导入 V2rayN 客户端的步骤

1.  在工具页面上配置您的**自定义规则**和**默认策略**。
2.  点击 **"生成 JSON 配置"** 按钮。
3.  点击 **"复制 JSON 代码"** 按钮。
4.  打开 V2rayN 客户端，进入 **设置** -> **路由设置**。
5.  在路由设置中选择导入自定义规则或手动编辑，粘贴您复制的 JSON 代码即可。
